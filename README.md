# Catze Discord Bot Test Repo

## Tech Stack
### Server : NestJS + Prisma
### Client : multiple web + Discord Bot


## Directory Information
- `/bot` : discord bot
- `/prisma` : prisma ORM setting files
- `/src` : NestJS src files


## Installation

```bash
$ npm install
```

## Running the app

## Before Running 
- NestJS running not needs spcial .env files
- For Running Discord Bot, You should make `setting_variable.json` and should set specific env variables.


```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
